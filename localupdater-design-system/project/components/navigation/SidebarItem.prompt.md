Sidebar row for a folder, source, or category. Selected state shows an amber 2px left bar + tinted background.

```jsx
<SidebarItem
  icon={<Icon name="desktop" size={14} />}
  label="デスクトップ"
  count={12}
  selected
/>
<SidebarItem icon={<Icon name="folder" size={14} />} label="Dropbox" depth={1} count={8} />
```
