Color-coded square thumbnail for a file. Used as the leading glyph in every list row and grid tile.

```jsx
<FileTypeBadge ext="dwg" size={32} />
<FileTypeBadge kind="image" size={48} />
<FileTypeBadge ext="pdf" showLabel={false} size={20} />
```

Categories: cad / image / slides / doc / sheet / pdf / other. Extend by editing `FILE_KINDS` + `FILE_EXT_MAP` in the source.
