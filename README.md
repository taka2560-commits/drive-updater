# LocalUpdater

ローカルPC内の特定ディレクトリ（デスクトップ、ドキュメントなど）をスキャンし、「最近更新されたファイル」を一覧表示するデスクトップアプリケーションです。React と Electron を用いて構築されており、モダンなUIとネイティブなファイル操作を両立しています。

## 主な機能

- **3つのビューモード**
  - **リスト** — 時間バケット（今日・昨日・今週・先週）でグループ化されたファイル一覧。サイズ・更新日時・スター表示付き
  - **タイムライン** — 縦型タイムラインレイアウトのカード形式。日ごとのファイル変更を俯瞰
  - **カレンダー** — 月間ヒートマップ。日ごとの活動量を6段階の色濃度で表示し、サイドパネルに詳細一覧
- **3つのテーマ**
  - **Earth** — スチールブルー基調 + アンバーアクセント（デフォルト）
  - **Night** — ディープネイビー + ブライトブルー + イエロー
  - **Light** — ホワイト基調 + ミント + コーラル
- **ファイルの高速スキャンとリアルタイム監視**
  - `chokidar` によるファイル変更の即時反映
- **フォルダナビゲーション**
  - 標準フォルダ（デスクトップ・ドキュメント・ダウンロード）に加え、カスタムフォルダの追加が可能。ダブルクリックでドリルダウン、パンくずリストで階層移動
- **強力なフィルタリング**
  - 種類（Docs / Sheets / PDF / Image / Slides / CAD / Other）、期間（24時間〜1か月）、サイズ、キーワード検索を組み合わせて絞り込み
- **ファイル操作**
  - 右クリックコンテキストメニュー（全ビュー共通）: 開く、保存場所を開く、スター切替、パスをコピー、名前変更、ゴミ箱に移動
- **複数選択 + 一括操作**
  - Ctrl/Shift クリックで複数選択し、まとめてスター・ゴミ箱・パスのコピーが可能
- **プレビューと詳細メタデータ**
  - 画像はサムネイル、テキスト系ファイル（.txt/.md/.csv/.json/.log/ソースコード等）は内容をプレビュー。作成・更新・アクセス日時やサイズも表示
- **ファイルタイプバッジ**
  - 拡張子に応じた色分けバッジ（CAD/Image/Slides/Doc/Sheet/PDF/Other）

## 開発環境のセットアップ

### 前提条件
- Node.js (v18以降推奨)
- npm

### インストールと起動

```bash
# リポジトリのクローン
git clone https://github.com/taka2560-commits/drive-updater.git
cd drive-updater

# パッケージのインストール
npm install

# 開発モードで起動 (Vite + Electron)
npm run dev
```

### ビルド (実行ファイルの作成)

```bash
npm run build
```
成功すると、`release` フォルダ内に Windows 向けのインストーラー (`.exe`) が作成されます。

## テーマの切り替え

サイドバー下部のカラースウォッチをクリックするとテーマが切り替わります。選択は `localStorage` に保存され、次回起動時に復元されます。

## スキャン除外について

設定の「スキャン除外」で、一覧から除外したいフォルダ名・ファイル名を登録できます（既定: `node_modules`, `.git`, `.DS_Store`, `dist`）。
照合は**名前の完全一致**で行われるため、`dist` を登録しても `distribution.pdf` のような無関係なファイルは除外されません。

## プロジェクト構成

```
src/
├── components/       # React コンポーネント
│   ├── Toolbar.tsx          # ビュー切替・検索バー
│   ├── Sidebar.tsx          # フォルダナビ・期間フィルタ・テーマ切替
│   ├── FileTable.tsx        # リストビュー
│   ├── FileGroupList.tsx    # タイムラインビュー
│   ├── CalendarView.tsx     # カレンダーヒートマップビュー
│   ├── ContextMenu.tsx      # 共有右クリックメニュー
│   ├── DetailPane.tsx       # ファイル詳細パネル
│   ├── FilterBar.tsx        # 種別フィルタチップ
│   ├── FileTypeBadge.tsx    # ファイル種類バッジ
│   └── ui.tsx               # Badge, Button, NavItem
├── theme/
│   ├── themes.css           # 3テーマのCSS変数定義
│   └── tokens.ts            # JS用テーマ値
├── lib/
│   ├── grouping.ts          # 時間バケットグルーピング
│   ├── heatmap.ts           # カレンダー集計
│   ├── fileType.ts          # 拡張子→アイコン・色マッピング
│   ├── format.ts            # 日時・サイズのフォーマット
│   └── activity.ts          # アクティビティバー計算
└── assets/fonts/            # Inter, Noto Sans JP, JetBrains Mono
```
