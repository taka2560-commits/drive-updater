# LocalUpdater (DriveUpdater)

ローカルPC内の特定ディレクトリ（デスクトップ、ドキュメントなど）をスキャンし、「最近更新されたファイル」を一覧表示するデスクトップアプリケーションです。React と Electron を用いて構築されており、モダンなUIとネイティブなファイル操作を両立しています。

## ✨ 主な機能

- **ファイルの高速スキャンとリアルタイム監視**
  - Node.js の `fs` と `chokidar` を使用し、ローカルの変更を即座にUIへ反映。
- **フォルダ／ファイルの一覧表示とドリルダウン**
  - 通常のファイルマネージャーのようにフォルダも表示。ダブルクリックで中に入り、**パンくずリスト**で上の階層へ戻れます。
- **対象フォルダの切り替えと追加**
  - 標準のデスクトップ・ドキュメント・ダウンロードに加え、任意のカスタムフォルダをスキャン対象に追加可能。
- **3種類のレイアウト**
  - A案（標準: サイドバー＋テーブル＋詳細ペイン）／B案（タイムライン・カードビュー）／C案（カレンダーヒートマップ＋モーダル詳細）を設定から切り替え。
- **強力なフィルタリング**
  - 種類（Docs / Sheets / PDF / Image / Slides / CAD / Other）・**期間（今日・今週・今月）**・**サイズ（1MB+ / 10MB+ / 100MB+）**・キーワード検索を組み合わせて絞り込み。
- **ファイル操作**
  - **ゴミ箱へ移動（復元可能）**・**インラインのリネーム**・右クリックのコンテキストメニューに対応。
- **複数選択＋一括操作**
  - ⌘/Ctrl・Shift クリックで複数選択し、まとめてスター・ゴミ箱・パスのコピーが可能。
- **プレビューと詳細メタデータ**
  - 画像はサムネイル、**テキスト系ファイル（.txt/.md/.csv/.json/.log/ソースコード等）は内容**をプレビュー。作成・更新・アクセスの時間情報やサイズも表示。
- **スター（お気に入り）機能**
  - 頻繁にアクセスするファイルをスター付けして素早くアクセス可能。
- **OSネイティブ機能との連携**
  - ファイルを直接開いたり、保存場所のエクスプローラーを起動することが可能。

## 🛠 技術スタック

- **Electron 42** — デスクトップシェル／ネイティブファイル操作
- **React 19 + TypeScript 6** — UI
- **Vite 8** — ビルド／開発サーバー
- **lucide-react** — アイコン
- **chokidar** — ファイル監視
- スタイリングは **CSS変数によるテーマトークン**（Earth / Night / Light の3テーマ）

## 🚀 開発環境のセットアップ

### 前提条件
- Node.js (v18以降推奨)
- npm

### インストールと起動

```bash
# リポジトリのクローン
git clone https://github.com/taka2560-commits/drive-updater.git
cd drive-updater

# パッケージのインストール
npm install

# 開発モードで起動 (Vite + Electron)
npm run dev
```

### ビルド (実行ファイルの作成)

```bash
npm run build
```
成功すると、`release` フォルダ内に Windows 向けのインストーラー (`.exe`) が作成されます。

## ⚙️ スキャン除外について

設定の「スキャン除外」で、一覧から除外したいフォルダ名・ファイル名を登録できます（既定: `node_modules`, `.git`, `.DS_Store`, `dist`）。
照合は**名前の完全一致**で行われるため、`dist` を登録しても `distribution.pdf` のような無関係なファイルは除外されません。
